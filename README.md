# 🔐 Análise de Logs de Segurança com Power BI

Projeto desenvolvido com foco em análise de eventos de segurança, simulação de monitoramento e identificação de padrões suspeitos.

## 🎯 Objetivo
Analisar logs simulados para identificar comportamentos anômalos, como tentativas de acesso não autorizado e variações de atividade ao longo do tempo.

## 📊 Dashboard

![Dashboard](dashboard.png)

## 🔍 Análises realizadas

- Identificação de IP com múltiplas falhas de login (possível brute force)
- Distribuição de eventos (login, download, upload)
- Análise temporal de eventos de segurança

## 🛠️ Tecnologias utilizadas

- Microsoft Power BI
- Análise de dados
- Logs simulados

## 🧠 Conclusão

O projeto demonstra como a análise de dados pode ser aplicada à segurança da informação, auxiliando na identificação de padrões suspeitos e monitoramento de eventos.
